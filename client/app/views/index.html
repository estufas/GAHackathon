Find Me With Lee

An application that stores article links and titles in a PostgreSQL database. Scrapes pages for information, has basic search functionality, Bycript secured user sign up and log in, allowing users to save favorite articles and post/edit/delete comments about them. Server built using Node.js and Express, with Javacsript, JQuery and BootStrap for the front end.

Parse

Parse is a tool to help language learners read in another language and find important vocabulary to study. The idea is backed by the Pareto Principle: 20% of the words in a text give 80% of it's understanding. Built with Ruby on Rails, javaScript and JQuery front end functionality, and Foundation for the front end. This was a group project that included four people, two people working on the front end for styling leaving myself and another person to code the backend.

Chess Master

Chess application allowing users to play live games. MEAN stack, using the Socket.IO plug in for chat and live play, and user secured log in with JSON web tokens. While there wasn't anything original or innovative about this project it provided us with a useful learning opportunity, in dealing with live socket connections and the way information is passed between the server and client. This was a two person project with shared front and back end responsibilities.

Ruby Spell Checker

Spell checker built using Ruby and Ruby on Rails, without the use of a spell checker plug in or dictionary API. The application takes user input, uses a binary search to find a match, if not match is found words with the shortest edit distance are returned to the user as suggested corrections.
